# The International Utilities Package

The International Utilities Package provides routines for dealing with sorting, currency, measurement systems, and date and time formatting. It is important that you use the routines in this package, rather than the System Utility routines contained in the Macintosh ROMâ€”the ROM routines are not as accurate and (because they are used by the File system) they can't be localized for different countries.
