<span id="page-182-1"></span>If your application provides functions for formatting text, you can include a **Format menu** as a top-level menu or as a submenu of the Edit menu. It may be appropriate to group some items that are in the Format menu into submenus, such as Font, Text, or Style.  
<span id="page-182-8"></span><span id="page-182-5"></span>**Figure 13-18** A Format menu  
![](images/_page_182_Picture_12.jpeg)  
**Show Fonts (Command-T).** The first item in the Format menu should be Show Fonts, which displays the Fonts window.  
<span id="page-182-4"></span>**Show Colors (Command-Shift-C).** Displays the Colors window.  
<span id="page-183-4"></span>**Bold (Command-B).** Boldfaces the selected text or toggles boldfaced text on and off. If used as a toggle, indicate when it is on by having a checkmark next to the command in the menu.  
<span id="page-183-9"></span>**Italic (Command-I).** Italicizes the selected text or toggles italic text on and off. If used as a toggle, indicate when it is on by having a checkmark next to the command in the menu.  
<span id="page-183-12"></span><span id="page-183-3"></span>**Underline (Command-U).** Underlines the selected text or toggles underlined text on and off. If used as a toggle, indicate when it is on by having a checkmark next to the command in the menu.  
<span id="page-183-6"></span>**Bigger (Command–Shift–equal sign).** Causes the selected item to increase in size in defined increments.  
<span id="page-183-10"></span>**Smaller (Command-hyphen).** Causes the selected item to decrease in size in defined increments.  
<span id="page-183-11"></span>**Copy Style (Command-Option-C).** Copies the style—font, color, and size for text—of the selected item. It may be appropriate to put this item in a Style submenu along with related items.  
<span id="page-183-1"></span>**Paste Style (Command-Option-V).** Applies the style of one object to the selected object.  
<span id="page-183-13"></span><span id="page-183-5"></span>**Align Left (Command-{).** Left-aligns a selection.  
<span id="page-183-2"></span>**Center (Command-|).** Center-aligns a selection.  
<span id="page-183-14"></span>**Justify.** Evenly spaces a selection.  
<span id="page-183-7"></span>**Align Right (Command-}).** Right-aligns a selection.  
**Show Ruler.** Displays a formatting ruler.  
<span id="page-183-15"></span><span id="page-183-8"></span>**Copy Ruler (Command-Control-C).** Copies formatting settings such as tabs and alignment for a selection to apply to a another selection and stores them on the Clipboard.  
<span id="page-183-0"></span>**Paste Ruler (Command-Control-V).** Applies formatting settings (that have been saved to the Clipboard) to the selected object.
