<span id="page-47-9"></span>When the user moves an item by dropping its icon on a folder icon, the dropped icon disappears and the highlighting is removed from the destination folder icon.  
<span id="page-47-3"></span>If an icon represents a task, such as printing, you may want to provide progress feedback to indicate that the task is being carried out.
