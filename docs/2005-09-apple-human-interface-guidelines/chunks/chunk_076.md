<span id="page-63-4"></span>Everyapplication defines aworkflow, even if the taskit performs is simple.From the user's perspective, however, workflow also encompasses tasks the user might perform using different applications, separately or in conjunction.  
Be aware that many users employ automated workflow solutions that rely on the scriptability of applications. If your users are likely to value scriptability, be sure to provide it (see ["Know](#page-24-2) Your [Audience](#page-24-2) " (page 25) for more information on determining the needs of your users).  
<span id="page-63-1"></span>Mac OS X version 10.4 introduced two ways for users to manage workflows. One is Automator, a tool that lets users create complex workflows that combine discrete processes from many different applications. The other is Dashboard, a way to access and keep track of many separate tasks at the same time. The following sections describe these technologies.
