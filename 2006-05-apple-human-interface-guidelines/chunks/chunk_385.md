The location at which the user releases the mouse button when selecting a range of objects.  
#### **active window**  
<span id="page-322-3"></span>A window that applies to the user's current task. Active windows are distinguished from inactive windows by the look of the title bar and the window controls. Multiple windows can be active simultaneously. See also "key [window"](#page-327-1) (page 328); "main [window"](#page-327-2) (page 328).
