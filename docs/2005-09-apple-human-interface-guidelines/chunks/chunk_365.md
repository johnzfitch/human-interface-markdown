<span id="page-314-4"></span>A modeless window that displays an application's version and copyright information.  
#### **accumulating attribute group**  
A set of attribute choices in which the user can select multiple items, such as Bold and Italic. See also ["mutually](#page-319-0) exclusive attribute group" (page 320).  
#### **action button**  
The button that confirms the message text in a dialog. The action button is in the lower right corner of a dialog. It is often, but not always, the default button.
