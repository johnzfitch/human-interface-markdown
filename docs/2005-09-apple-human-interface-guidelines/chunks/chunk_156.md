<span id="page-112-4"></span>When an item is being dragged, your application must determine whether to scroll the contents or allow the item to "escape" the window. If your application allows items to be dragged outside of windows,you should define an automatic scrollingregion. Automaticallyscroll a destinationwindow only if it is also the source window and is frontmost. Don't automatically scroll inactive windows.
