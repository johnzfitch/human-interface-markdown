A control that creates a token out of a user's text input.  
#### **toolbar**  
A collection of buttons at the top of a window just belowthe title bar. A toolbar can be hidden orrevealed with a toolbar button in the title bar.  
#### **tool palette**  
A collection of buttons and other controls in a utility window.  
#### **type-ahead**  
Queuing of keystrokes for processing later. It occurs when the user types faster than the computer can handle or when the computer is unable to process the keystrokes.  
#### <span id="page-322-1"></span>**user state**  
A window's user-defined size and position. See also ["standard](#page-321-1) state" (page 322); "zoom [button"](#page-322-2) (page 323).  
#### <span id="page-322-0"></span>**utility window**  
A window that floats above other windows and provides tools or controls that users can work with while documents are open. See also ["document](#page-317-2) [window"](#page-317-2) (page 318).  
#### **view font**  
The default font used in text and lists. This may be user adjustable, as it is in the Finder.
