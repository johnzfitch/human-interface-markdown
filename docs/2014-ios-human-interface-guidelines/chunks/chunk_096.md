A **tab bar** gives people the ability to switch between different subtasks, views, or modes in an app.  
![](images/_page_149_Picture_7.jpeg)  
**API Note:** A tab bar is contained in a tab bar controller, which is an object that manages the display of a set of custom views. To learn more about defining a tab bar in your code,see "Tab Bar Controllers" and "Tab Bars".
