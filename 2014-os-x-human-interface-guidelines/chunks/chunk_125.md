A **color well** indicates the current color of the selected object and, when clicked, displays the Colors window, in which users can specify a color. To learn more about the Colors window,see Colors and Fonts [Windows](#page-267-0) (page 268).  
<span id="page-202-1"></span>**API Note:** To define a color well in your code, use the NSColorWell class.  
Multiple color wells can appear in a window. For example, Rulers preferences in Keynote contains two color wells that allow users to change the colors of guides.
