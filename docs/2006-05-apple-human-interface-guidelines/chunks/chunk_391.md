#### **application menu**  
<span id="page-322-2"></span>A menu that contains items that apply to the application as a whole, rather than to a specific document or otherwindow. The application menu for the current active application appears immediatelyto the right of the Apple menu.  
#### **application-modal dialog**  
A dialog that prevents the user from performing any operations within the owner application other than those in the dialog. See also ["document-modal](#page-325-1) [dialog"](#page-325-1) (page 326); ["sheet"](#page-328-0) (page 329).
