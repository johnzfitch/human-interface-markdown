Text in a dialog that users can't modify.  
#### **submenu**  
A menu that descends from another menu. The title of the submenu is a menu item in the parent menu. See also ["hierarchical](#page-318-2) menu" (page 319).  
#### **system font**  
The font used for text in menus and in modeless dialogs, and for titles of document windows. It is 13-point Lucida Grande Regular.  
#### **tab view**  
A control that provides a convenient way to present information in a multipane format.  
#### **text input field**  
A rectangular area in which the user enters text or modifies existingtext. Also called an editable text field, it supports keyboard focus and password entry.
